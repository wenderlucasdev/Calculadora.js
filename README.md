# Calculadora 

Calculadora funcional desenvolvida com HTML, CSS e JavaScript, apresentando um design moderno com troca de temas (Light/Dark) e efeitos visuais de brilho (glow).

## Funcionalidades
- Operações matemáticas básicas.
- Interface responsiva (funciona em celular e PC).
- Alternância de tema com persistência visual.
- Efeitos de brilho dinâmicos no ícone de tema.

## Tecnologias
- **HTML5** (Estrutura)
- **CSS3** (Grid, Flexbox e Variáveis)
- **JavaScript** (Lógica de cálculo e manipulação de DOM)
- **Google Fonts** (Material Icons)
